## 2025-06-18 - Python PyYAML Safe Loading Bottleneck
**Learning:** PyYAML's `yaml.safe_load()` uses the pure Python `SafeLoader`, which is heavily used across this codebase for document conversions (especially the `yaml_to_json_structure` parsing) and is very slow. The benchmark shows `yaml.safe_load` takes 2.0541s for 1000 parsing operations, while `yaml.load(yaml_str, Loader=yaml.CSafeLoader)` takes only 0.2534s, an ~8x speedup.
**Action:** Replace `yaml.safe_load(str)` with a centralized fast loading function that leverages the C-extension `CSafeLoader` when available, falling back to `SafeLoader`. Evaluate the `try/except` import at module level to avoid import overhead. Implement this utility in `utils/yaml_converter.py` and import it wherever `yaml.safe_load` is used to boost performance globally.
