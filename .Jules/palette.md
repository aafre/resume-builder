## 2024-05-18 - Replacing Unicode Buttons with Accessible Icons in SectionControls
**Learning:** Found an existing `SectionControls` component using unicode arrows and trashcan characters for buttons without labels or accessible focus. This is a common pattern for "invisible" utility buttons in rapid-prototyping, but creates major accessibility barriers (no screen-reader context) and inconsistent visual rendering across OS platforms.
**Action:** When replacing basic unicode buttons in utility rows, always migrate them to a standardized SVG library (like `react-icons`), add explicit `aria-label`s, `title`s (for tooltips), and `type="button"`. Apply semantic focus states (`focus-visible`) and distinct interactive states (e.g., `text-red-600 hover:bg-red-50` for destructive actions) to improve keyboard navigation and usability.
